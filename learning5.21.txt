3.css画对勾
首先给他一个外部的div画圆，并且相对父级div在其角落
html
<div class="rt_circle" *ngIf="class.checked">
                    <a class="checks"></a>
                </div>
css
.rt_circle  {
        position: relative;
        display: inline-block;
        left: 42px;
        bottom: 10px;
        width: 18px;
        background-color: green;
        height: 18px;
        border-radius: 50%;
    }
然后在其内部画对勾，使用两个"/"加上旋转来实现
.checks {
        display: inline-block;
        position: relative;
        left: -4px;
        top: -6px;
        width: 6px;
        height: 2px;
        background: white;
        line-height: 0;
        font-size: 0;
        vertical-align: middle;
        -webkit-transform: rotate(45deg);
    }

    .checks:after {
        content: '/';
        display: block;
        width: 11px;
        height: 2px;
        background: white;
        -webkit-transform: rotate(-90deg) translateY(15%) translateX(50%);
    }
